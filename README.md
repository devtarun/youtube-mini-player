# YouTube Mini Player

😋️ This electron app runs 📹️ YouTube videos in mini desktop view.

## Installation
Uses Yarn as a dependency manager

```
git clone https://github.com/devtarun/youtube-mini-player.git

yarn install
```

### Running
```
yarn start
```

### Building
```
yarn make
```
